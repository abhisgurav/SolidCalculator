# SolidCalculator

A simple console calculator implemented in C# demonstrating **SOLID principles**.

## Features
- Addition, Subtraction, Multiplication, Division
- Follows **SOLID design principles**
- Open for extension (add new operations easily)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/abhisgurav/SolidCalculator.git
2.	Open the solution in Visual Studio. 
3.	Build and run the project. 
4.	Follow on-screen prompts to perform calculations. 

SOLID Principles Applied
•	SRP: Each operation has a single responsibility. 
•	OCP: Easily extendable for new operations. 
•	LSP: All operations can be used interchangeably. 
•	ISP: Simple interface IOperation. 
•	DIP: Calculator depends on abstraction, not concrete implementations. 

Example
Enter First Number:
10
Enter Second Number:
5
Choose operation (+, -, *, /):
*
Result: 50
